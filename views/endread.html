ending

So, here we are! To be honest, I never really liked computers, but back then I realized that I can make a better usage of it. I can use it to build and express myself, I just started to love it.

Programming is really learning you how to think in a different way with different logic. It’s a lifestyle. Quickly I started to be creative again, because I already went through a process, that learned me so much, especially about the dip period. I gained that different desire for constant learning and growth in a different way of spirituality. Programming is an art and I think that everyone should learn, not just how to program, but how to create and build. Here are some books that I recommend => The dip (written by Godin), Program or be programmed (written by Rushkoff), Think like a programmer (written by Spraul).

I hope that I didn’t make a bad impression on you, dear reader.
